# KatLeskova
<p>Данная программа выполняет некоторые арифметические операции над двумя введёнными числами.</p> 
<img src="http://gospodaretsva.com/wp-content/uploads/2011/11/4- 
21.png"/> 
<br> 
<h3>Функции</h3> 
<ul> 
<li>Нахождение суммы\разности двух чисел</li> 
<li>Нахождение произведения\частного двух чисел</li> 
<li>Нахождение факториалов двух чисел</li> 
<li>Возведение каждого числа в степень другого</li> 
<li>Определение членов ряда Фибоначчи</li> 
</ul>
